# Fraction
A Fraction implementation built from scratch using Python's magic methods.

## Features

- Automatic simplification
- Operator overloading
- Comparisons
- Reverse operators
- Type conversion

## Installation

```bash
pip install fraction
```

## Example

```python
from fraction import Fraction

a = Fraction(1,2)
b = Fraction(3,4)

print(a+b)
```
